# Mafia Telegram Bot (aiogram 3.9.0)

Минимальный бот для игры в Мафию в Telegram-группах.

## Возможности

- Создание лобби в групповом чате
- Подключение и выход игроков
- Назначение ролей (Мафия/Мирный)
- Отправка роли каждому игроку в личные сообщения

## Стек

- Python 3.11+
- aiogram 3.9.0

## Быстрый старт

1. Создай бота через @BotFather и получи токен.
2. В корне проекта создай `.env` на основе `.env.example`.
3. Установи зависимости:

```bash
pip install -r requirements.txt
```

4. Запусти бота:

```bash
python main.py
```

## Команды бота

- `/start` - помощь
- `/panel` - кнопки управления регистрацией
- `/create` - создать лобби (только в группе)
- `/join` - войти в лобби
- `/leave` - выйти из лобби
- `/lobby` - показать текущих игроков
- `/begin` - начать игру
- `/roles` - список ролей и их описаний
- `/action` - сделать ход в личке бота
- `/kill` - открыть меню ночного действия в личке
- `/heal` - открыть меню ночного действия в личке
- `/check` - открыть меню ночного действия в личке
- `/night_end` - завершить ночь
- `/vote` - открыть меню дневного голосования в личке
- `/day_end` - завершить день

Во время игры бот автоматически отправляет живым игрокам в личку меню доступных действий на фазу.

Реализованы способности ролей: Маньяк (ночной удар), Любовница (блок роли на ночь), Бомж (ночной отчет-наблюдение), Самоубийца (личная победа при дневной казни), Камикадзе (забирает с собой игрока при казни).
- `/close` - закрыть лобби

## Важно

Перед раздачей ролей каждый игрок должен написать боту в личку `/start`, иначе бот не сможет отправить ему роль.
После старта каждый игрок получает в личке карточку своей роли с авторским описанием.

### Регистрация через кнопки

Открой в группе `/panel` и используй кнопки:

- `Запустить регистрацию`
- `Продлить регистрацию`
- `Завершить регистрацию и начать`
- `Отменить регистрацию`
